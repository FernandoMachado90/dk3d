# DK3D

An OpenGL 3D remake of the original Donkey Kong arcade game, released in 1981.

## Requisites
- OpenGL and GLUT ([instructions](https://cs.lmu.edu/~ray/notes/opengl/))

## Commands

### `make build`
Builds the game binary.

### `make run`
Starts the game locally.
